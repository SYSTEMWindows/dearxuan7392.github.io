Minecraft插件

极限追杀&死亡交换

Minecraft插件

使用Java编写，由国外我的世界著名主播Dream独创的插件：ManHunt-2和DeathSwap反编译而来，进行了汉化. 并在原有基础上增加了新功能.

极限追杀:
所有的猎人都会得到一个指南针，手持指南针右键即可追踪逃亡者.如果逃亡者在下界，则会指向下界传送门.
/hunter add <name> :添加猎人
/hunter remove <name> :移除猎人

死亡交换:
每隔一定时间交换双方的位置.
/deathswap add <name1> <name2> :添加玩家
/deathswap remove <name> :移除玩家
/deathswap settime <time> :设置时间间隔(单位:秒)
/deathswap start :开始游戏
/deathswap start :停止游戏
